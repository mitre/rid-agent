Overview

This section contains an assortment of tools that provide mechanisms for interacting with the RID Agent.

API Tools

  • Build Incoming Message API Query - A tool to help design a query for the Get Incoming Message API Service.
  • Build Outgoing Message API Push - A tool to help build the POST data for the Push Outgoing Message API Service.

Message Queue Tools

  • Add Incoming Message - Provides a mechanism for adding new RID Messages to the incoming message store.
  • Add Outgoing Message - Provides a mechanism for adding RID Messages to the outgoing message store.
  • Send RID Message - Provides a mechanism for sending an arbitrary RID Message to an arbitrary peer.
  • View/Delete Incoming Messages - Provides a mechanism for viewing the incoming message store.
  • View/Delete Outgoing Messages - Provides a mechanism for viewing the outgoing message store.

Other Tools

  • Check Certificate Chains - Provides a mechanism to analyze client certificate chains
